# Chunk 32: Comment Bubble Softening

Status: complete

## Goal

Make post-repair comments feel less intrusive in the browser game.

## Completed

- Shortened the reward dialogue after a win.
- Shortened and tightened staggered reaction bubbles so they no longer keep talking over the player for several seconds.
- Made dialogue and reaction bubbles translucent.
- Added face avoidance zones around the player and robot so bubbles prefer side/high/low placements instead of sitting on faces.
- Let the continue prompt appear sooner after a win.
- Bumped browser module version tags for the changed HUD and repair-flow modules.

## Notes

- This keeps a little celebration feedback, but it should feel much quieter and easier to ignore while playing.
